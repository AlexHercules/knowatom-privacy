隐私政策

KnowAtom - 智能对话助手

最后更新日期：2025年10月20日

引言

欢迎使用 KnowAtom（以下简称"本应用"）。本应用由 cute Hercules 开发和运营。 我们非常重视您的隐私保护和个人信息安全。本隐私政策将帮助您了解：

  • 我们收集哪些信息
  • 我们如何使用这些信息
  • 您如何管理自己的信息

请您在使用本应用前仔细阅读并理解本政策。如果您不同意本政策的任何内容，请停止使用本应用。

1. 我们收集的信息

1.1 您主动提供的信息

  • API 密钥：您在使用本应用时需要配置第三方 AI 服务商（如 OpenAI、Claude 等）的 API 密钥
  • 对话内容：您与 AI 助手的对话历史记录
  • 应用设置：您的个性化设置和偏好配置

1.2 自动收集的信息

  • 设备信息：设备型号、操作系统版本（仅用于界面适配）
  • 应用使用数据：文件访问时间戳、本地存储空间使用情况

重要说明：本应用不会收集您的位置信息、通讯录、相册、或任何其他设备敏感权限。

2. 信息的存储与使用

2.1 本地存储

所有数据（包括 API 密钥、对话记录、应用设置）均存储在您的设备本地，我们不会将这些数据上传到任何服务器。

  • 使用 iOS/iPadOS 系统的本地存储（UserDefaults、FileSystem）
  • 数据随应用卸载自动删除
  • 不进行云端备份或同步

2.2 第三方 AI 服务

当您使用本应用与 AI 对话时，您的对话内容将通过您配置的 API 密钥发送到相应的第三方 AI 服务商（如 OpenAI、Anthropic 等）。 这些服务商有各自的隐私政策，请您在使用前阅读：

  • OpenAI： https://openai.com/privacy
  • Anthropic： https://www.anthropic.com/privacy
  • Google AI： https://policies.google.com/privacy

注意：我们无法控制第三方服务商如何处理您的数据，建议您在使用前了解其隐私政策。

3. 信息的共享与披露

我们不会与任何第三方共享、出售或出租您的个人信息，除非：

  • 获得您的明确同意
  • 法律法规要求披露
  • 保护本应用或用户的合法权益

4. 数据安全

我们采取以下措施保护您的数据安全：

  • 本地加密：敏感数据（如 API 密钥）存储在 iOS 安全存储区（Keychain）
  • HTTPS 传输：所有网络请求均使用 HTTPS 加密传输
  • 无服务器架构：不存在中央服务器被攻击的风险

5. 您的权利

关于您的个人信息，您拥有以下权利：

  • 访问权：您可以随时在应用内查看您的所有数据
  • 删除权：您可以在应用内删除任何对话记录或导出的文件
  • 修改权：您可以随时修改您的 API 配置和应用设置
  • 导出权：您可以将对话导出为 Markdown 文件保存到本地

如何删除所有数据：卸载本应用即可删除所有本地存储的数据。

6. 儿童隐私保护

本应用不面向 13 岁以下儿童。我们不会有意收集儿童的个人信息。 如果您发现儿童向我们提供了个人信息，请通过下方联系方式告知我们。

7. 第三方 SDK 使用说明

本应用使用以下第三方 SDK，但不用于数据收集或追踪：

  • Capacitor：用于跨平台开发框架（Ionic Team）
  • React：用于用户界面开发（Meta Platforms）

本应用不包含任何广告 SDK、数据分析 SDK 或用户追踪工具。

7.1 支持的 AI 服务商

本应用支持以下 AI 服务商（您可以自行选择使用）：

  • DeepSeek、OpenAI、Anthropic (Claude)、Google Gemini
  • Moonshot、通义千问、智谱AI、百川智能、MiniMax

您的 API 密钥和对话数据仅在您的设备与您选择的服务商之间传输，我们不参与此过程。

8. AI 生成内容免责声明

本应用提供的所有 AI 对话功能均由第三方 AI 服务商提供。 AI 生成的内容可能不准确、不完整或包含错误，请您谨慎使用， 并在重要决策前核实信息的准确性。我们不对 AI 生成内容的准确性、可靠性承担责任。

9. 数据保留

由于所有数据均存储在您的设备本地，数据保留期限完全由您控制：

  • 对话记录：永久保留在您的设备上，直到您手动删除或卸载应用
  • API 密钥：存储在 iOS Keychain 中，直到您删除或卸载应用
  • 应用设置：保留在本地，直到您重置或卸载应用

我们不会在任何服务器上保留您的数据。

10. 隐私政策的更新

我们可能会不时更新本隐私政策。更新后的政策将在应用内发布，并在顶部标注更新日期。 如有重大变更，我们会通过应用内通知的方式告知您。
