❓ 常见问题

Q1: 如何开始使用 KnowAtom？

首次使用需要配置 AI 服务的 API 密钥：

  1. 打开应用，点击底部"设置"按钮
  2. 选择要使用的 AI 服务商
  3. 输入对应的 API 密钥（可从服务商官网获取）
  4. 点击"验证连接"确保配置正确
  5. 返回主界面即可开始对话
Q2: 支持哪些 AI 服务商？

KnowAtom 目前支持以下 AI 服务商：

  • DeepSeek - 高性价比的国产 AI 服务
  • OpenAI - GPT-3.5, GPT-4 系列
  • Claude - Anthropic 的 Claude 系列
  • Gemini - Google 的 Gemini 系列
  • Moonshot - 月之暗面 Kimi
  • Qwen - 阿里通义千问
  • 智谱 AI - GLM 系列
  • 百川智能 - Baichuan 系列
  • MiniMax - MiniMax 系列
Q3: 如何获取 API 密钥？

不同服务商的 API 密钥获取方式：

  • DeepSeek: 访问 platform.deepseek.com，注册并在控制台创建 API Key
  • OpenAI: 访问 platform.openai.com，需要国际信用卡
  • 其他服务商: 访问对应官网注册并获取密钥

提示：API 密钥仅存储在您的设备本地，不会上传到任何服务器。

Q4: 什么是树状对话模式？

树状对话模式允许您从任意对话节点创建新的分支，探索不同的对话方向：

  • 在任意消息上左滑或长按，选择"从此分支"
  • 创建新分支后，可以尝试不同的提问方式
  • 所有分支都会保留，方便对比和回顾
  • 点击消息可以切换到不同的分支
Q5: 我的对话数据存储在哪里？

所有对话数据都存储在您的设备本地，包括：

  • 对话历史记录
  • API 密钥配置
  • 应用设置

隐私保护：KnowAtom 不会将任何数据上传到我们的服务器。您的对话内容仅在调用 AI 服务时发送到对应的 AI 服务商。

Q6: 如何导出对话记录？

导出对话非常简单：

  1. 在对话列表中，长按要导出的对话
  2. 选择"导出对话"
  3. 选择导出格式（Markdown 或文本）
  4. 通过系统分享功能发送或保存
Q7: 应用需要网络连接吗？

是的，KnowAtom 需要网络连接来访问 AI 服务。但是：

  • 浏览历史对话不需要网络
  • 导出对话不需要网络
  • 只有发送新消息时才需要联网
Q8: 遇到"连接失败"怎么办？

请检查以下几点：

  1. 网络连接：确保设备已连接到互联网
  2. API 密钥：确认密钥正确且未过期
  3. 账户余额：确认 AI 服务账户有足够余额
  4. 服务状态：检查 AI 服务商是否正常运行
  5. API 地址：如使用自定义地址，确认地址正确
Q9: 应用收费吗？

应用本身完全免费，无需支付任何费用。

但是，使用 AI 服务需要向对应的 AI 服务商付费：

  • 大部分服务商提供免费额度供新用户试用
  • 超出免费额度后需要充值
  • 费用由 AI 服务商收取，与 KnowAtom 无关
Q10: 如何删除对话记录？

删除单个对话：

  • 在对话列表中左滑对话项
  • 点击"删除"按钮

清空所有数据：

  • 进入"设置"
  • 滑到底部找到"清除所有数据"
  • 确认操作（此操作不可恢复）
